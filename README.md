## Deployment on Hostinger

### Method 1: Using File Manager (Recommended)

1. Log in to your Hostinger account
2. Go to "Hosting" → "Manage"
3. Click on "Files" → "File Manager"
4. Navigate to the `public_html` folder
5. Upload all website files (index.html, styles.css, script.js, contact-form-handler.php) to the `public_html` folder
6. Make sure all files are in the root directory and not in a subfolder
7. Ensure file permissions are set correctly (644 for files, 755 for folders)

### Method 2: Using FTP

1. Download and install an FTP client (e.g., FileZilla)
2. Get your FTP credentials from Hostinger
3. Connect to your server using the FTP client
4. Navigate to the `public_html` folder
5. Upload all website files to the `public_html` folder
6. Make sure all files are in the root directory and not in a subfolder

### Important Notes

- Ensure all files are uploaded to the correct directory (`public_html`)
- Make sure file names are exactly as they are in your local project (case-sensitive)
- Verify that PHP is enabled on your hosting account
- Test the contact form to ensure it works properly
- Check that all links and paths are correct

### Troubleshooting

If the website appears unorganized or elements are misplaced:

1. Check that all files (index.html, styles.css, script.js) are uploaded to the correct directory
2. Verify file names match exactly (case-sensitive)
3. Clear your browser cache
4. Check the browser console for any errors
5. Ensure CSS and JavaScript files are properly linked in the HTML file
